# id-last-digit-date

This Java program selects a specific date based on the last digit of a Turkish ID number.  
It uses a `switch-case` structure to match the digit and print the corresponding date.

## Features
- Reads the last digit of the ID
- Uses switch-case logic
- Prints a date based on the digit
- Includes a default case for invalid input
- Beginner-friendly and console-based

## How to Run
1. Open the Java file
2. Run the program
3. See the date based on the last digit
